# Rich Reporter

## Usage

```shell
$ python3 -m vedro -r rich -vv
```

**Verbose Levels**

| Verbose | Show Scenario | Show Steps| Show Exception | Show Scope (scenario variables) |
|:--------|:-------------:|:---------:|:--------------:|:----------:|
|      |✅| | | |
|`-v`  |✅|✅| |
|`-vv` |✅|✅|✅| |
|`-vvv`|✅|✅|✅|✅|
